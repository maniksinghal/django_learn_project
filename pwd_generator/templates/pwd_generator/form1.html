Password generator

Below we have a form that will generate a 6 or 7 length password.
The action field of the form refers to a password_url variable.
The variable points to a 'my_password' path in utils.py, which maps to a def password: in views.py
The interesting thing is that there is no actual 'my_password' path.
It will show up in the address-bar of the browser but it is just a mapping of the requested page to the corresponding python function.
